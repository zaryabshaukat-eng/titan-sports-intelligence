# TITAN — Sports Intelligence OS

Enterprise-grade sports intelligence operating system combining market data, statistics, and explainable AI.

## Stack

- **Runtime:** Bun
- **Framework:** TanStack Start (SSR) + React 19
- **Styling:** Tailwind CSS v4 + custom design tokens (oklch colour space)
- **Charts:** Recharts
- **Routing:** TanStack Router (file-based, auto-generated routeTree)
- **Resizable panels:** react-resizable-panels v4.x (`Group`, `Panel`, `Separator`)
- **Config:** `@lovable.dev/vite-tanstack-config` wraps Vite + Nitro

## How to run

```
bun run dev   # starts the Vite dev server on port 5000
```

The workflow **Start application** is pre-configured.

## Project structure

```
src/
  routes/              # File-based TanStack Router pages
  components/
    titan/             # App-specific components
      AppShell.tsx     # Sidebar, topbar, command palette, notifications, live activity
      CommandPalette.tsx
      NotificationCenter.tsx
      LiveActivityCenter.tsx
      ConfidenceWidgets.tsx
      DataTable.tsx
      primitives.tsx
    ui/                # shadcn/ui components
  styles.css           # Tailwind v4 theme + utilities
  routeTree.gen.ts     # Auto-generated by TanStack Router plugin (also manually kept in sync)
```

## Key implementation notes

- **react-resizable-panels v4.x** uses `Group`/`Panel`/`Separator` (not `PanelGroup`/`PanelResizeHandle`) and `orientation` not `direction`.
- **routeTree.gen.ts** is auto-regenerated when the dev server runs and picks up new route files. It was manually patched to add `/research` and `/system-status` before first start.
- The hydration mismatch warning from TanStack devtools (`data-tsd-source`) is cosmetic and harmless.
- All data is placeholder/mock — no external API connections are required to run.

## Features added

1. **Global Command Palette** (Ctrl+K) — navigate pages, teams, leagues, actions
2. **Notification Center** — sliding panel with categories, filtering, mark-read
3. **Live Activity Center** — real-time metrics panel with flickering live data
4. **Research Workspace** (`/research`) — three-column resizable layout with markdown editor
5. **System Status** (`/system-status`) — enterprise infrastructure health page
6. **Enhanced AI Intelligence** — per-engine CPU/memory/latency/uptime metrics
7. **Confidence Widgets** — ConfidenceGauge, ProbabilityRing, RiskMeter, SignalStrength, etc.
8. **DataTable** — generic sortable/filterable/paginated table with CSV export
9. **Mobile-responsive sidebar** — hamburger menu for small screens
